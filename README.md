# 🍰 Página Web de Pastelería

## 📌 Descripción del proyecto

Este repositorio contiene el desarrollo de una **página web de una pastelería**, creada como parte de la **Actividad RA4**, aplicando el uso de **GitHub** para el control de versiones. El proyecto presenta una estructura básica de un sitio web orientado a la promoción de productos de repostería, mostrando información clara y visualmente atractiva para los clientes.

La página está diseñada para representar una pastelería, destacando sus productos, servicios y una presentación amigable para el usuario.

---

## 🎯 Objetivo

El objetivo principal del proyecto es:

* Aplicar correctamente el uso de **Git** para el control de versiones.
* Utilizar **GitHub** como repositorio remoto.
* Desarrollar una página web sencilla para una pastelería.
* Documentar los cambios mediante *commits*.

---

## 🧁 Contenido de la página

La página de la pastelería incluye:

* Presentación del negocio
* Información sobre productos de repostería
* Diseño básico y estructurado
* Archivos organizados para su fácil comprensión

---

## 🛠️ Tecnologías utilizadas

* HTML
* CSS
* GitHub
* Visual Studio Code

---

## 🔄 Control de versiones

El repositorio cuenta con un historial de *commits* que refleja:

* La creación inicial del proyecto (*Initial commit*)
* La carga de archivos correspondientes a la página web

Esto permite llevar un registro claro de los cambios realizados durante el desarrollo.
